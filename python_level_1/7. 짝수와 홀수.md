# 7. 짝수와 홀수

### ✨문제

> 정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.
>
> ##### 제한 조건
>
> - num은 int 범위의 정수입니다.
> - 0은 짝수입니다.
>
> ex) num = 3 / return "Odd"
>
> ex) num = 4 / return "Even"



#### 🎈 내 풀이

```python
def solution(num):
    return num%2 and 'Odd' or 'Even'
#0 = false, 1 = true
```
